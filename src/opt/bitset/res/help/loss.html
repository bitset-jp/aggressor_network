loss - Loss Packet Set an independent loss probability for the outgoing packets from the selected network interface. Lost packets are not transmitted. USAGE: loss PERCENT (%) ARGUMENTS: PERCENTpacket loss rate (%) EXAMPLES: 0 > loss 10% ... set loss 0*> update... Write to NIC --- 22:24:43 --- [0] eth0: qdisc netem 8009: root refcnt 6 limit 1000 loss 10% NOTICE: Execute update command to reflect on the NIC.
